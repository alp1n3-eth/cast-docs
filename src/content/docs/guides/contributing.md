## CLA Assistant

The [CLA Assistant](https://cla-assistant.io) is used to ensure compliance with the current license that Cast uses.

## Pre-Contribution

Before doing the items below:

1. Code is written.
2. A pull request is opened.

Check to see if there is an existing:

1. Issue.
2. Discussion.
