## Methods

### Brew

<Steps>
1. Open your favorite terminal.

2.  Run the following command:

        ```bash
        brew install cast
        ```

</Steps>

### Building It

<Steps>
1. Clone the repo.

2.  Run the following command:

        ```bash
        go build
        ```

</Steps>

## Verify the Install

<Steps>
1. Run the `cast` command.

2.  The output should look something like this:

        ```
        NAME: Cast - Make sending HTTP requests & testing APIs EZPZ.
           Submit any bugs as an issue in the repo: https://github.com/alp1n3-eth/cast/issues
        ```

</Steps>
